Instructions

  • Create a new repository (see: man git-init)
  • Add & commit this file to that repo
  • Create a new feature branch named feat-color
  • Create another new feature branch named feat-position
  • switch to the feat-color branch and change the color of the circle to blue
  • merge the feat-color branch into master with with a merge commit (it cannot be a fast-forward merge)
  • switch to the feat-position branch and move the circle down 50 pixels
  • merge the feat-position branch into master with a merge commit
  • both sets of changes should be reflected in master
  • delete both feature branches
  • create a log file of the commits with the command log --oneline --graph --decorate --all > a02.log.txt
  • submit both index.html and a02.log.txt in a zip file named git01.zip
